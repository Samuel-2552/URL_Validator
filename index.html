URL Validator

About:

The current version 1.2 supports upto 500 URL's at a stretch to be validated, i.e. whether the site is accessible or not and whether the keywords entered by the user is present in that site or not.

Finally after validation of the provided URL's it gives the option to download as excel sheet.

It is still under development, new features will be released soon that will include:

  1. Crawling into the links.
  2. File upload for URL's.
  3. Improvise the verbosity of comments

Where it all started?

This all started with a task that was assigned to me to validate all the URL's for proceeding with pentesting. I felt it very boring and repetitive so wrote a script for myself to complete the work. Then I felt that my temamtes are also struggling to do the same task, so I shared the code with them, but only few utilized it. So I came up with generalizing the code and developed a frontend UI to ease the task even more.

Later I found out that this a continuous process in cybersecurity and thought to develop it even more so that it can be used by everyone. The current version 1.2 has some drawbacks that needs to be worked upon.

I'm looking for contributors, who can contribute to this project to make it a success. Feel free to ping if you're interested in collaborating towards this project.

Benefits of using this:

  • It automates your task, where it takes a day or so to validate, it will take less than an hour.
  • It reduces man power, where it takes 5 or more persons to complete the validation task, it will finish it off with a single click.
  • Saves Money, Time and Man Power.
